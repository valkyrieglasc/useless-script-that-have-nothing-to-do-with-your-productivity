kitty as wallpaper (background):
```
kitten panel --output-name <name> --margin-<top / left / right / bottom> <number> --edge=background --focus-policy not-allowed --layer background command 
```
EX:
> kitten panel --output-name kitty-wallpaper --margin-top 40 --edge=background --focus-policy not-allowed --layer background cava
